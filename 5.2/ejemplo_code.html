Este es un ejemplo de cómo usar la etiqueta <code> junto con <pre> para mostrar un bloque de código formateado:


        function factorial(n) {
            if (n === 0) {
                return 1;
            } else {
                return n * factorial(n - 1);
            }
        }

        console.log(factorial(5)); // Imprimirá 120
